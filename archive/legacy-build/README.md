# Legacy Build Files

This directory contains archived Ant build files from the original 2010-2011 development.

## Files

- `module_spellingwordlistsetup.xml` - Ant build configuration for SpellingWordListSetup module
- `module_spellingwordlistsetup_Wed_Jan_12_21-37-30_MST_2011.xml` - Timestamped backup of Ant build file

## Historical Context

These files were used with:
- Apache Ant build system
- Google App Engine Java SDK 1.3.8 (from 2010)
- IntelliJ IDEA project structure from 2011

## Current Build System

The project now uses:
- Maven (pom.xml files)
- Modern GWT Maven Plugin
- Java 17 (server) / Java 11 (GWT client)

These legacy files are preserved for historical reference only and are not used in the current build process.
